# 金融材料智能解析与风险洞察

## 概述

本 Skill 用于对中文金融文档（包括但不限于企业年报、半年报、季报、券商研报、上市公告、招股说明书等）进行结构化解析，自动提取关键财务指标，识别潜在风险信号，分析文本情绪倾向，并生成标准化的风险分析报告。

适用场景：

- 投资者快速了解上市公司财务健康状况
- 风控人员批量筛查公告中的风险线索
- 研究人员从财报中提取结构化数据进行横向对比
- 金融从业人员快速生成研报摘要与风险评估

## 触发条件

当用户输入满足以下任一条件时，本 Skill 自动激活：

1. **关键词触发**：用户消息中包含以下关键词之一：
   - 财报分析、年报分析、半年报分析、季报分析
   - 风险分析、风险评估、风险识别
   - 研报解读、研究报告分析
   - 财务指标提取、关键指标分析
   - 上市公司公告分析、招股书分析
   - 投资风险评估、信用风险评估

2. **数据特征触发**：用户粘贴的文本中同时出现以下至少 3 项财务数据特征：
   - 营业收入 / 营收
   - 净利润 / 归母净利润
   - 资产负债率 / 负债率
   - 经营现金流 / 现金流量
   - 毛利率 / 净利率
   - 同比增长 / 环比增长

## 操作步骤

### 第一步：文档预处理

1. 识别用户提供的金融文档文本类型（年报/半年报/季报/研报/公告/招股书）
2. 提取报告期（如"2025年度"、"2025年第三季度"）
3. 识别报告主体（公司名称、股票代码）
4. 清洗文本中的噪音数据（页眉页脚、格式标记等）

### 第二步：关键财务指标提取

使用 `scripts/financial_indicator_extractor.py` 脚本辅助提取以下指标：

| 指标类别 | 具体指标 |
|---------|---------|
| 收入指标 | 营业收入、营收同比增长率、营收环比增长率 |
| 利润指标 | 净利润、归母净利润、毛利率、净利率、利润同比增长率 |
| 负债指标 | 总资产、总负债、资产负债率、流动比率、速动比率 |
| 现金流指标 | 经营活动现金流净额、投资活动现金流净额、筹资活动现金流净额 |
| 效率指标 | 总资产周转率、应收账款周转率、存货周转率 |
| 每股指标 | 基本每股收益、稀释每股收益、每股净资产 |

### 第三步：风险信号识别

依据 `reference.md` 中的风险信号知识库，逐项检查偿债能力风险、盈利能力风险、现金流风险、经营风险、合规与治理风险、文本情绪分析六大维度。

### 第四步：风险评级

| 评级 | 标准 |
|------|------|
| **高风险** | 存在 3 个及以上高风险信号 |
| **中风险** | 存在 1-2 个高风险信号，或 3 个及以上中风险信号 |
| **低风险** | 仅有低风险信号，且关键财务指标处于健康区间 |

### 第五步：生成报告

按照 `templates/risk-report.md` 模板生成结构化风险分析报告。

## 已知限制

1. **数据准确性依赖输入**：本 Skill 仅对用户提供的文本进行分析，无法验证数据的真实性或完整性。
2. **行业差异化不足**：当前版本采用通用财务分析框架，对特定行业个性化指标覆盖不够深入。
3. **情绪分析局限**：情绪分析基于关键词频率和语义模式，可能无法准确捕捉含蓄或反语表达。
4. **不构成投资建议**：本 Skill 输出的风险分析报告仅供学习和研究参考。
5. **语言支持**：当前版本仅支持中文金融文档。
6. **时效性**：风险信号阈值和行业基准参考值需根据最新市场环境调整。
7. **非标准格式处理**：对于扫描件 OCR 识别后的文本等非标准格式，指标提取准确率可能下降。
