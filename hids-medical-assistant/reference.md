## 四、常见危险信号（红旗征象）

以下症状组合提示可能存在危及生命的情况，应立即就医或拨打 120：

| 危险信号组合 | 可能疾病 | 建议 |
|--------------|----------|------|
| 胸痛 + 大汗 + 呼吸困难 + 恶心 | 急性心肌梗死 | 立即拨打 120，勿自行就医 |
| 突发一侧肢体无力/麻木 + 言语不清 | 急性脑卒中 | 立即拨打 120，记录发病时间 |
| 突发剧烈头痛 + 呕吐（"雷击样"） | 蛛网膜下腔出血 | 立即拨打 120 |
| 呼吸困难 + 喘息 + 不能平卧 | 急性左心衰 | 立即急诊 |
| 高热 > 39°C + 意识模糊 + 寒战 | 严重感染/脓毒症 | 立即急诊 |
| 腹痛 + 板状腹 + 血压下降 | 消化道穿孔/腹膜炎 | 立即急诊 |
| 咯血 + 呼吸困难 | 肺栓塞/大咯血 | 立即急诊 |
| 意识丧失 / 抽搐 | 癫痫/心脑血管意外 | 立即拨打 120，保护气道 |

## 五、常见检验指标组合异常的临床意义

| 异常组合 | 临床提示 | 建议关注 |
|----------|----------|----------|
| FPG 升高 + HbA1c 升高 + 尿糖阳性 | 糖尿病 | 内分泌科，糖化血红蛋白达标 |
| TC 升高 + LDL-C 升高 + TG 升高 | 高脂血症 | 心内科，评估心血管风险 |
| FPG 升高 + TG 升高 + HDL-C 降低 + 腹型肥胖 | 代谢综合征 | 综合评估心血管及糖尿病风险 |
| ALT 升高 + AST 升高 + GGT 升高 | 肝功能损害 | 消化内科/肝病科，排查病因 |
| Cr 升高 + BUN 升高 + eGFR 降低 | 肾功能损害 | 肾内科，评估慢性肾脏病分期 |
| UA 升高 + 关节疼痛 | 痛风 | 风湿免疫科，降尿酸治疗 |
| WBC 升高 + NEUT% 升高 + CRP 升高 | 细菌感染 | 感染科/相关科室 |
| WBC 降低 + Hb 降低 + PLT 降低 | 血细胞三系减少 | 血液科，排除骨髓问题 |
| TSH 升高 + FT4 降低 | 甲状腺功能减退 | 内分泌科 |
| TSH 降低 + FT4 升高 | 甲状腺功能亢进 | 内分泌科 |
| cTnI 升高 + 心电图缺血改变 | 心肌损伤 | 心内科/急诊，排查心梗 |
| BNP/NT-proBNP 升高 | 心力衰竭 | 心内科 |
| D-二聚体显著升高 + 呼吸困难 | 肺栓塞可能 | 呼吸内科/急诊 |
