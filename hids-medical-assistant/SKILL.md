# 医疗健康智能理解与辅助决策 Skill

> 科大讯飞 AI 开发者大赛 HIDS 赛道参赛作品

## 一、触发条件

| 类别 | 关键词示例 |
|------|------------|
| 体检报告类 | 体检报告、化验单、检查结果、指标异常、血常规、肝功能、肾功能、血脂、血糖 |
| 病历解读类 | 病历解读、出院小结、门诊记录、诊断书 |
| 症状分析类 | 症状分析、身体不适、头疼、胸闷、腹痛、乏力、发热 |
| 科室推荐类 | 挂什么科、科室推荐、看什么医生 |

## 二、操作步骤

### 场景 A：病历结构化
1. 识别病历文本中的结构化段落
2. 使用 `scripts/medical_entity_extractor.py` 提取医疗实体
3. 将非结构化病历转换为结构化格式

### 场景 B：体检报告解读
1. 识别检验指标名称、实测值、参考范围
2. 按轻度/中度/重度偏离分级
3. 使用 `templates/health-report.md` 模板生成报告

### 场景 C：症状分析与科室推荐
1. 从用户描述中提取症状实体
2. 参照 `reference.md` 症状-科室映射表匹配
3. 识别危险信号（红旗征象），评估紧急程度

## 三、引用支撑文件

| 文件 | 用途 |
|------|------|
| `templates/health-report.md` | 体检报告结构化输出模板 |
| `scripts/medical_entity_extractor.py` | 纯 Python 正则医疗实体提取工具 |
| `reference.md` | 检验指标正常范围、症状-科室映射 |
| `examples/medical-record-sample.md` | 模拟门诊病历样本 |
| `examples/diagnosis-output-sample.md` | 结构化分析输出样本 |

## 四、已知限制

1. 仅能处理文本形式的医疗信息
2. 检验指标参考范围为成人通用值
3. 仅支持中文医疗文本

## 五、医疗免责声明

**本 Skill 提供的所有分析结果仅供参考，不构成任何形式的医疗诊断、治疗方案或用药指导。** 如遇身体不适或紧急情况，请立即前往正规医疗机构就诊或拨打急救电话 120。
